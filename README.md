# 50zo
En este mini proyecto se desarrollará el juego llamado "Cincuentazo". Es un juego de cartas de Poker donde los jugadores (humano y máquina) deben sobrevivir utilizando sus cartas. Se jugará contra 1, 2 o 3 jugadores máquina; donde cada jugador siempre debe tener una mano de 4 cartas que podrá jugar en su turno
